# AI Study Notes Generator 🧠

AI Study Notes Generator is a simple web application that converts long paragraphs into short and easy-to-read study notes.

Users can paste text, click the generate button, and instantly get summarized bullet-point notes.

---

## Features

- Paste long study material
- Generate quick notes
- Bullet-point summary format
- Clean and modern UI
- Copy notes button
- Responsive design

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## How to Run

1. Download or clone this repository
2. Open the project folder
3. Run `index.html` in your browser
